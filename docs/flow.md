# Flow

HTTP-запрос проходит через Pydantic-валидацию и нормализацию текста. Затем сервис выбирает провайдера: mock по умолчанию или Ollama при `LLM_PROVIDER=ollama`. Результат провайдера валидируется Pydantic-моделью и возвращается как JSON. При ошибке Ollama сервис пишет предупреждение в лог и использует mock fallback.
